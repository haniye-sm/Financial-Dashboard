# PowerBI Financial Dashboard

<img width="80%" alt="Screenshot" src="https://github.com/user-attachments/assets/ce09858e-aaa9-4d59-be55-b709b72e4e71">


This repository contains a dynamic financial dashboard built using Power BI. The dashboard provides an insightful comparison of key financial metrics between two years (2019 and 2020), allowing users to quickly identify trends, changes, and areas for deeper analysis.


## Tools Used
-	Data Cleaning – Excel - Power Query
- Preparation & Processing   Power Query - Dax Query
-	Data Visualization – Power BI


## Key Features

- **Revenue**: Displays the total revenue for the current and previous year, with percentage change and trend visualization.
- **Cost of Goods Sold (COGS)**: Monitors COGS with a clear indication of the percentage change.
- **Gross Profit**: Shows the gross profit with comparative analysis for both years.
- **Expenses**: Summarizes operating expenses and highlights any increase or decrease.
- **EBIT (Earnings Before Interest and Taxes)**: Displays EBIT values with clear percentage change visualization.
- **Finance Costs**: Tracks finance-related expenses with significant percentage change alerts.
- **Net Income Before Tax**: Compares pre-tax income for both years.
- **Tax**: Provides details on tax variations year-over-year.
- **Net Income**: Shows the bottom-line net income with year-over-year comparison.
- **Net Income Margin**: Visualizes the net income margin as a percentage, offering a quick understanding of profitability.

## Visual Elements

1. **Matrix Table**: A detailed table that lists key financial categories such as revenue, expenses, and net income, comparing current and previous year data with percentage change.
2. **KPI Cards**: For each key metric (Revenue, Gross Profit, EBIT, etc.), KPI cards display the current and previous year's values, percentage change, and a trend graph for quick visual insights.
3. **Donut Chart**: A donut chart visualizes the net income margin, offering an easy-to-understand percentage of net income in relation to revenue.
